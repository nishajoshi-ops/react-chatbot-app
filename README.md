# react-chatbot-app
A chatbot application built with React.js featuring real-time messaging UI, async bot responses, loading state, and auto-scroll functionality.


## 🚀 Features

- 💬 User and chatbot message interface
- ⚛️ Built with React components
- 🔄 Dynamic message updates using useState
- ⏳ Loading state while bot responds
- ⌨️ Send messages using Enter key
- 📜 Auto-scroll chat using useRef and useEffect
- 🎨 Custom styled chat UI

## 🛠️ Technologies Used

- React.js
- JavaScript (ES6+)
- HTML
- CSS
- Babel
- Day.js
